[~] 2FA-012 — Autenticazione a due fattori (TOTP)
------------------------------------------------------------------------------
  Priorita: MEDIA
  Dipendenze: USER-007 (pagina impostazioni necessaria per setup)

  DESCRIZIONE:
  Aggiungere supporto TOTP per aumentare la sicurezza dell'accesso.
  Compatibile con Google Authenticator, Authy, ecc.

  DETTAGLI TECNICI:
  Backend:
    1. Installare otplib e qrcode
    2. Aggiungere a schema.prisma User: totpSecret String?,
       totpEnabled Boolean @default(false)
    3. Endpoint POST /api/user/2fa/setup → Genera secret + QR code URI
    4. Endpoint POST /api/user/2fa/verify → Verifica codice e abilita
    5. Endpoint POST /api/user/2fa/disable → Disabilita (richiede password)
    6. Modificare login: se totpEnabled, richiedere codice TOTP dopo email/password
  Frontend:
    1. Sezione 2FA nella SettingsPage
    2. Mostrare QR code durante setup (libreria qrcode.react)
    3. Input codice 6 cifre per verifica
    4. Step aggiuntivo nel LoginPage se 2FA abilitato

  FILE COINVOLTI:

    MODIFICARE: server/prisma/schema.prisma (totpSecret, totpEnabled on User)
